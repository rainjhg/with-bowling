📋 앱스 스크립트 URL 만드는 방법 (약 5분)
1
새 탭에서 script.google.com 접속 → 새 프로젝트 클릭 → 편집창 기존 코드 전체 선택 Ctrl+A 후 삭제
2
아래 코드 복사하기 버튼 클릭 → 빈 편집창에 붙여넣기 Ctrl+V → 저장 Ctrl+S → 프로젝트 이름: 볼링동호회
불러오는 중...
3
오른쪽 상단 배포 버튼 클릭 → 새 배포 → ⚙️ 아이콘 → 웹 앱 선택
4
• 다음 사용자로 실행: 나
• 액세스 권한: 모든 사용자 ← 반드시!
→ 배포 → 팝업에서 액세스 승인 → 내 계정 선택 → 고급 → 볼링동호회(으)로 이동 → 허용
5
배포 완료 화면에서 웹 앱 URL 복사 https://script.google.com/macros/s/…/exec → 위 입력창에 붙여넣기 → 연결하고 시작하기!
